# Estrategia de Marketing y Ventas — Sara Studio

Instrucciones detalladas de marketing para el agente.

## Cuándo usar esta skill
- Usa esta skill al mostrar precios, escribir textos comerciales (copy) o diseñar el flujo de conversión.
- Es útil para garantizar que el sitio web funcione como un embudo de ventas efectivo.

## Cómo usarla

### Psicología de Precios
- **Comparativa de Mercado (Efecto Anclaje)**: En lugar de enlaces externos (que fugan ventas), muestra una lista de precios de la competencia en texto:
  `❌ Liverpool: $4,290`
  `❌ Palacio de Hierro: $4,500`
  `✅ Sara Studio: $1,800`
- Formatea los precios sin decimales (ejemplo: `$1,500` y no `$1,500.00`).
- El precio de venta NUNCA debe superar el 50% del precio de mercado.

### Señales de Confianza (Trust Signals)
- Muestra la comparativa de precios de tiendas reconocidas como referencia.
- Muestra insignias (badges) de "Nuevo con etiquetas".
- Las imágenes deben mostrar claramente las etiquetas de precio originales.

### Urgencia
- Usa "🔥 Última pieza" solo si es realmente cierto.
- Usa "✨ NUEVO" para productos recientes.

### Flujo de Conversión
- Mantén el botón de WhatsApp prominente.
- Prellena el mensaje de WhatsApp para reducir la fricción de compra:
  `Hola! 👋 Me interesa un producto de Sara Studio: 🛍️ [MARCA] - [PRODUCTO] 💰 Precio: $[PRECIO] MXN ¿Está disponible?`
